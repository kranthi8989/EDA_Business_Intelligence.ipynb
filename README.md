## 🎯 Objectives

- Perform descriptive statistical analysis.
- Explore sales patterns and customer behavior.
- Answer business questions using SQL queries.
- Identify trends and relationships in the dataset.
- Create visualizations for business intelligence.
- Design a dashboard mock-up for decision-making.

---

## 📂 Dataset

The dataset contains cafe sales transaction records including:

- Product Name
- Quantity Sold
- Unit Price
- Total Spent
- Payment Method
- Transaction Date
